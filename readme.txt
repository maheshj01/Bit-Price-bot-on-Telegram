this bot gives you the statistics of cryptocurrencies in real time,All data is fetched from the Bittrex Exchange
inorder to test this code you need to do the following

1. fetch your bot token from botfather on Telegram and paste it in bot.py on line no 6
2. inorder to run the bot you need to download pyTelegrambotApi module
    a)through the terminal you can install the module using the command pip install pyTelegramBotApi
    b)once the module is installed your environment is ready to run the bot
3. you can now run the script and you will find the bot running

note: the code is not yet fully developed, so you may find some instabilities
